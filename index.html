Le VS

Sur notre site web, nous allons vous presenter deux jeux bien différents entre eux : Field of Mistria qui est un jeu de simulation d'agriculture et Clair Obscur qui est un jeu RPG aux combats tour par tour punitif. A la suite de votre lecture, vous pourrez voter via le formulaire en bas de base quel jeu vous pensez être le plus attrayant !

Puisqu'il faut bien commencer quelque part, quel paysage vous attire le plus entre ces deux là ? Cliquez dessus !
